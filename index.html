My name is Annette Brecheisen and this website is my final project for the Digital Skills for Visual Media class at City College of San Francisco. In this class we learned the basics of raster and vector graphics, time-based media and print/web layout. All of my projects are connected, each one building on the project before. I began with the theme of escapism and imagination, expressed through a fairytale lense.
